# Assignment2
Life Hack or Urban Myth App (For First-Time Moms)
Overview
The Life Hack or Urban Myth App is an androis application developed using Kotlin in Android Studio.

This version of the app is specifically designed for first-time pregnant women, helping them distinguish between common pregnancy myths and safe, practical life hacks.

The app uses an interactive flashcard quiz to educate users in a simple, engaging and supportive way.

Purpose
The purpose of this application is to: 
- Help first-time mothers identify safe vs unsafe advice
- Reduce the spread of harmful pregnancy myths
- Improve decision-making and awareness
- Provide practical, real-life pregnancy tips
- Create an easy and engaging learning experience

Design Considerations
1. User Interface (UI)
   - Clean and calming design
   - Soft, readable colours suitable for a maternal theme
   - Large, clear buttons for ease of use
2. User Expirience (UX)
   - Simple navigation for all users
   - Encouraging and supportive feedback
   - Clear distinction between myths and facts
3. Functionality
   - Loop through all flashcard questions
   - Track correct answers using a score counter
   - Provide feedback using conditional logic
4. Accessibility and Sensitivity
   - Content is designed to be supportive and non-judgmental
   - Avoids overwhelming or harmful information
   - Focuses on safe and practical advice
